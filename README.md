## Overview
This project is a Kotlin-based technical task designed to showcase the fundamental principles of Kotlin development, including but not limited to: object-oriented programming, functional programming, and using Kotlin to build efficient applications.

## Features
- **Kotlin Language**: Utilizing the modern features of Kotlin language.
- **Data Handling**: Efficient data processing and transformation practices.
- **Concurrency**: Basic understanding and implementation of concurrency in Kotlin.

## Getting Started
### Prerequisites
- Kotlin 1.6 or later
- JDK 11 or later
- An IDE such as IntelliJ IDEA or Android Studio.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/LilithUwU/TecnicalTaskEM.git
   ```
2. Navigate into the project directory:
   ```bash
   cd TecnicalTaskEM
   ```
3. Open the project in your chosen IDE.
4. Build the project to ensure all dependencies are resolved.

## Usage
You can run the application using the IDE or via command line with:
```bash
./gradlew run
```
